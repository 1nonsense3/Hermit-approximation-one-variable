# C++ Hermite Interpolation

Реализована программа для приближения заданной функции интерполяционным полиномом Эрмита на выбранном отрезке. Программа вычисляет значения функции и её производной в узлах интерполяции, определяет коэффициенты полинома, оценивает погрешность приближения и строит графики исходной и приближённой функций с помощью Gnuplot.

## Возможности

* выбор интервала интерполяции;
* выбор количества узлов;
* вычисление значений функции в равномерно расположенных узлах;
* численное вычисление производной;
* построение интерполяционного полинома Эрмита;
* вычисление значения полинома в заданной точке;
* оценка погрешности приближения;
* построение графиков исходной функции и её приближения.

## Структура проекта

* `main.cpp` — точка запуска программы, обработка аргументов командной строки и подготовка данных;
* `aprox1.cpp` — построение коэффициентов интерполяционного полинома Эрмита и вычисление его значений;
* `utility.cpp` — реализация исходных функций и численного вычисления производной;
* `plot.cpp` — построение графиков с использованием Gnuplot;
* `Header.h` — объявления функций проекта.

## Поддерживаемые функции

Параметр `k` определяет функцию, которую необходимо приблизить:

* `0` — (f(x) = 1);
* `1` — (f(x) = x);
* `2` — (f(x) = x^2);
* `3` — (f(x) = x^3);
* `4` — (f(x) = x^4);
* `5` — (f(x) = e^x);
* `6` — (f(x) = \frac{1}{25x^2 + 1}).

## Требования

Для компиляции и запуска необходимы:

* компилятор с поддержкой C++;
* Gnuplot для отображения графиков.

Установка Gnuplot в Ubuntu или Debian:

```bash
sudo apt install gnuplot
```

Установка Gnuplot в macOS через Homebrew:

```bash
brew install gnuplot
```

## Компиляция

С помощью `g++`:

```bash
g++ main.cpp aprox1.cpp plot.cpp utility.cpp -o approximation
```

## Запуск

Программа принимает четыре аргумента:

```bash
./approximation <левая_граница> <правая_граница> <количество_узлов> <номер_функции>
```

Пример:

```bash
./approximation -1 1 10 6
```

В данном примере программа:

* рассматривает отрезок `[-1, 1]`;
* использует `10` узлов интерполяции;
* приближает функцию (f(x) = \frac{1}{25x^2 + 1}).

После запуска программа выводит значение исходной функции и интерполяционного полинома в контрольной точке, показывает величину погрешности и открывает график сравнения функций.

## Принцип работы

Узлы интерполяции равномерно распределяются по заданному отрезку. Для каждого узла вычисляются значения исходной функции и её производной, после чего методом разделённых разностей строится интерполяционный полином Эрмита.

Полученный полином вычисляется по схеме Горнера. Для визуального сравнения программа передаёт значения исходной и приближённой функций в Gnuplot и отображает их на одном графике.
